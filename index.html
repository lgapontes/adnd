Gerador de personagens AD&D 2ed


Sobre este gerador
Este é um gerador aleatório de personagens para Advanced Dungeons and Dragons 2ª edição. Ele leva como base os livros Livro do Jogador, Ravenloft: Domínios do Medo, The Complete Psionics Handbook e o DarkSun Rules Book (1991). Note que há algumas restrições em relação às classes e raças (ou seja, algumas classes não podem ser assumidas por certas raças). Isso é refletido automaticamente nos campos de seleção, caso você opte em forçar a rolagem com algumas classe ou raça em particular.

Como usar este gerador?
O botão Exportar PDF imprime um arquivo PDF contendo os dados gerados, de forma que você possa compartilhar sua ficha e guardá-la para uso posterior. Observe que ao atualizar esta página, a ficha rolada (e apresentada na tela) será perdida. O campo Campanha Ravenloft serve para ajustar a habilidade Poder da Fé dos Clérigos e Anacoretas às condições estabelecidas no livro Ravenloft: Domínios do Medo (enfraquecendo as mesmas). O campo Campanha Dark Sun serve para ajustar os atributos às condições estabelecidas no livro DarkSun Rules Book (1991) (fortalecendo os mesmos), entre outros elementos presentes neste cenário. O campo Usar Multiclasse ou Classe Dupla vai habilitar as opções de Multiclasse (apenas para as raças semi-humanas) ou definir os requisitos mínimos de atributos das Classes Duplas (apenas para as raças humanas). Mesmo que o nível escolhido seja superior ao 1, este rolador não permite escolher a segunda classe da Classe Dupla - ele só aplica os valores necessários para que o jogador escolha a posteriori.
